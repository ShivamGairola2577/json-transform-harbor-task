# JSON Age Filter Task

You are given a JSON file at:

/app/input.json

The file contains a list of users with the structure:

[
  {"name": "Alice", "age": 25},
  {"name": "Bob", "age": 17}
]

Your task:

1. Read /app/input.json
2. Filter users with age >= 18
3. Convert their names to uppercase
4. Sort the names alphabetically
5. Write the result to:

/app/output.txt

Each name must be on a new line.

Example output:

ALICE
CHARLIE
